# Styling DayPicker

Change the appearance of the HTML elements composing DayPicker, such as headings, cells, buttons.

## Default style

DayPicker comes with for a minimal, lightweight style. Import it into your app's root component, or add it into your build pipeline.

```tsx filename="App.tsx" {2}
import './my-app.css';
import 'react-day-picker/dist/style.css';

export default App() {
  // snip
}
```

This will add the [DayPicker stylesheet](https://github.com/gpbl/react-day-picker/blob/main/style.css) into your app. The stylesheet includes some CSS global variables, to override colors and sizes:

```css
.rdp {
  --rdp-cell-size: 40px;
  --rdp-accent-color: #0000ff;
  --rdp-background-color: #e7edff;
  /* Switch to dark colors for dark themes */
  --rdp-accent-color-dark: #3003e1;
  --rdp-background-color-dark: #180270;
  /* Outline border for focused elements */
  --rdp-outline: 2px solid var(--rdp-accent-color);
  /* Outline border for focused and selected elements */
  --rdp-outline-selected: 2px solid rgba(0, 0, 0, 0.75);
}
```

## Styling Modifiers

In DayPicker, change the style of [Modifiers](/basics/modifiers) with the `modifiersClassNames` or `modifiersStyles`. This will set the class name or the inline style for the days with the modifiers.

<SourceCode fileName="styling-modifiers.tsx" />

import StylingModifiers from 'react-day-picker/docs/examples/styling-modifiers.tsx';

<RenderingBox>
  <StylingModifiers />
</RenderingBox>

With CSS modules, pass the generated class name instead:

```tsx {7-10,1}
import style from './your-style.module.css';

function App() {
  return (
    <DayPicker
      mode="multiple"
      modifiersClassNames={{
        selected: style.selected,
        today: style.today
      }}
    />
  );
}
```

## Styling with CSS

This approach involves just CSS and works well if you need the same style across your app.

<Steps>
### Import a new CSS file

Create and import it after the default style, e.g. `my-date-picker.css`:

```tsx
import 'react-day-picker/dist/style.css';
import './my-date-picker.css';
```

### Override the default CSS properties

Use the browser's DevTools or refer to the [stylesheet source](https://github.com/gpbl/react-day-picker/blob/main/style.css) to find the right selectors to override.

```css title="my-date-picker.css"
.rdp-day_today {
  color: red;
}
```

</Steps>

### Using CSS Modules

With CSS modules, import the generated class name from the CSS and pass them to the `classNames` prop.

<SourceCode fileName="styling-css-modules.tsx" />

import CSSModules from 'react-day-picker/docs/examples/styling-css-modules.tsx';

<RenderingBox>
  <CSSModules />
</RenderingBox>

### Using Inline Styles

To change the appearance of any DayPicker element via inline-styles use the `styles` prop.

<SourceCode fileName="styling-inline.tsx" />

import InlineStyling from 'react-day-picker/docs/examples/styling-inline.tsx';

<RenderingBox>
  <InlineStyling />
</RenderingBox>
